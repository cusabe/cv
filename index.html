Mobile App Development

I collaborated with a team of 4 using Gitlab to build a Java Android App to search for short stay accommodation. The front end included a visually appealing and responsive interface, Firebase login, query and sort order entry screen, listing and details screens, and help pages. The back end included an efficient binary search tree, text string tokeniser and parser to interpret search commands, and JSON data storage of simulated records. The project was a lot of fun, especially mocking up hotel records with Java Faker FunnyNames.
